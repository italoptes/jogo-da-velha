# Jogo da Velha em Python

Este projeto foi feito como parte da minha retomada aos estudos de **Python** .  
O objetivo foi relembrar conceitos básicos da linguagem, como estruturas de controle, listas, funções e lógica de programação.

## Sobre o jogo
O jogo é executado diretamente no terminal.  
Você escolhe se quer jogar com **X** ou **O**, e o computador faz jogadas aleatórias.

### Funcionalidades:
- Exibição visual do tabuleiro
- Validação de jogadas
- Verificação automática de vitória ou empate
- Opção de reiniciar o jogo após o fim da partida
